# Lab4-ML

In this lab, I worked on analyzing an Amazon dataset and applying basic data science and machine learning techniques. The goal was to understand the data, extract useful insights, and build a simple predictive model.

what I Did:
- Loaded the dataset into a Jupyter Notebook using Python  
- Cleaned the data by handling missing values and correcting data types  
- Explored the dataset to understand patterns and relationships between variables  
- Visualized the data using graphs to better interpret trends  
- Applied a machine learning model to make predictions based on the data  
- Evaluated the model to check its performance and accuracy  

Outcome:
Through this lab, I was able to gain practical experience in data preprocessing, visualization, and implementing a machine learning model. It also helped me understand how real-world datasets are handled and analyzed.
